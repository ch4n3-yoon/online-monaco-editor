## Execution
1. download
```bash
cd ~
git clone https://github.com/ch4n3-yoon/online-monaco-editor.git
```
2. build front codes
```bash
cd ~/online-monaco-editor/frontend/
yarn install
```
4. run contianers
```bash 
cd ~/online-monaco-editor/
sudo docker-compose up --build -d
```
